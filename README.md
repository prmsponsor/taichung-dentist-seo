## 🎯 差異化機會

1. **兒童牙科專頁 SEO** — KD=0，月搜量200，第一頁無兒童牙科專門診所
2. **週日/假日牙醫定位** — 僅新世代有明確主打，「台中週日牙醫」競爭低
3. **費用透明化頁面** — 所有競爭者均無清晰費用說明，是最明顯缺口
4. **睡眠牙科/止鼾牙套** — 僅京典美學有此服務頁，利基市場幾乎零競爭
5. **PTT/Dcard 口碑整合** — 社群信任度對看診決策影響大
6. **區域細分 Landing Page** — 北區、南屯、西屯等區域長尾詞 KD 普遍低

## 📋 內容缺口機會

1. **「台中牙醫診所推薦」深度長文**（KD=24，流量潛力900）— 1,500字+，加 FAQ schema
2. **費用比較頁面**（植牙月搜量1,900+、隱適美2,400+）— 所有競爭者都沒做好
3. **YouTube 衛教影片 SEO** — Video Thumbnail 版位競爭者極少，先行佈局
4. **Google 商家檔案 (GBP) 強化** — Local Pack 是最重要版位，每週發文+回覆評論
5. **全口重建/銀髮族照護深度指南**（月搜量1,000，KD=0）
